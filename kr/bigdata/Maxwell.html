# Maxwell

Maxwell 是 Java 编写的 MySQL 实时 数据抓取组件，实时读取 MySQL 二进制日志 Binlog，并生成 JSON 格式消息。

Maxwell 工作原理与 Canal 类似，将自己伪装成 MySQL Slave，从 MySQL Master 复制数据。

Maxwell 与 Canal 对比：

对比 Canal Maxwell
语言 Java Java
数据格式 格式自由 Json
采集数据模式 增量 全量/增量
数据落地 定制 支持 Kafka 等多种平台
HA 支持 支持
