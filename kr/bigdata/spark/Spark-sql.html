# Spark SQL

# SparkSQL 特点

  1. 易整合：无缝整合了SQL 查询和Spark 编程
  2. 统一数据访问：使用相同的方式连接不同的数据源
  3. 兼容 Hive：在已有的仓库上直接运行SQL/HiveSQL
  4. 标准数据连接：通过 JDBC/ODBC连接

# Hive ON Spark

  • 将 hive-site.xml 拷贝到 Spark conf/ 目录下
  • 把 Mysql 驱动拷贝到 Spark jars/ 目录下
  • 若访问不到 hdfs，则需要把 core-site.xml/hdfs-site.xml 拷贝到 Spark conf/ 目录下
  • 重启 spark-shell

# 数据抽象

  • DataFrame：是一种以 RDD 为基础的分布式数据集，类似于传统数据库的二维表，带有 Schema 元信息；DataFrame 表示为 DataSet[Row]，是弱类型，即 DataSet 的子集
  • DataSet：与 RDD 相比，保存了更多的描述信息，概念上等同于关系型数据库中的二维表；与 DataFrame 相比，保存了类型信息，是强类型的，提供了编译时类型检查
  • 共性：三者都是分布式弹性数据集；自动缓存运算；存在分区概念；都有惰性机制，遇到行动算子才会触发运算
  • 区别：RDD 不支持 SparkSQL，DataFrame 每行固定类型为 Row，DataFrame 是 DataSet 的一个泛型特例
  • 包含范围：DataSet => DataFrame => RDD

# 编程：DSL & SQL

  • 在 Spark 2.0 之前，SQLContext 是创建 DataFrame/DataSet 和执行 SQL 的入口；HiveContext 通过 hive sql 语句操作 hive 表数据，兼容 hive 操作，hiveContext 继承自 SQLContext
  • 在 Spark 2.0 之后，这些都统一于 SparkSession，SparkSession 封装了 SQLContext 及 HiveContext，实现了 SQLContext 及 HiveContext 所有功能；通过 SparkSession 还可以获取到 SparkConetxt

# WordCount

/**
 * Spark SQL & DSL WordCount
 */
object WordCountSparkSqlJob {

  val sparkConf: SparkConf = new SparkConf().setMaster("local[*]").setAppName("WordCountSparkSqlJob")
  val sparkSession: SparkSession = SparkSession.builder().config(sparkConf).getOrCreate()
  val sc: SparkContext = sparkSession.sparkContext

  import sparkSession.implicits._

  def main(args: Array[String]): Unit = {
    sc.setLogLevel("WARN")

    val fileDF: DataFrame = sparkSession.read.text("wc/WordCount.txt")
    val fileDS: Dataset[String] = sparkSession.read.textFile("wc/WordCount.txt")

    //    fileDF.printSchema()
    fileDS.printSchema
    //    fileDF.show()
    fileDS.show()

    wordCountSQL(fileDS)
    wordCountDSL(fileDS)

    sc.stop()
    sparkSession.stop()
  }

  /**
   * SQL：需要临时视图或全局视图来辅助
   *
   * @param fileDS
   */
  def wordCountSQL(fileDS: Dataset[String]): Unit = {
    val wordDS: Dataset[String] = fileDS.flatMap(_.split(" "))
    wordDS.createOrReplaceTempView("word_count")
    val sql =
      """
        |select value as word, count(value) as count
        |from word_count group by word order by count desc
        |""".stripMargin
    sparkSession.sql(sql).show()
  }

  /**
   * DSL：特定领域语言，不需要临时视图
   *
   * @param fileDS
   */
  def wordCountDSL(fileDS: Dataset[String]): Unit = {
    val wordDS: Dataset[String] = fileDS.flatMap(_.split(" "))
    wordDS.groupBy($"value".as("word")).count().orderBy($"count".desc).show()
  }

}
